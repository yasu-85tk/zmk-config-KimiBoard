## 1.デフォルトファームウェアについて 
>[!NOTE]
>**・Windowsではalt-ime-ahk、MacではKarabiner-Elementsを使用し、左右のAltキーで日本語・英語入力の切り替えを行う前提でキーを割り当てています。**

### 1-1.キーマップ  

####  通常のキーマップ  

| キー | 割り当て |
|:-|:-|
| 0キー| Bluetooth接続先変更（1→2→3のトグル） |
| 1キー| バックスペースキー|
| 2キー| スペースキー |
| 3キー| エンターキー |
| 1キー + 2キー| 左Altキー |
| 2キー + 3キー| 右Altキー |
| 1キー + 3キー| マウスボタンモードに切り替え（通常のキーマップとのトグル） |
| 1キー + 2キー + 3キー| 現在のBluetooth接続先情報を削除 |

####  マウスボタンモードのキーマップ  

| キー | 割り当て |
|:-|:-|
| 0キー| Bluetooth接続先変更（1→2→3のトグル） |
| 1キー| ホイールクリック|
| 2キー| 左クリック |
| 3キー| 右クリック |
| 1キー + 2キー| 左Altキー |
| 2キー + 3キー| 右Altキー |
| 1キー + 3キー| 通常のキーマップに切り替え（マウスボタンモードとのトグル） |
| 1キー + 2キー + 3キー| 現在のBluetooth接続先情報を削除 |

### 1-2.オートマウスレイヤー  

####  オートマウスレイヤー中のキーマップ  

| キー | 割り当て |
|:-|:-|
| 0キー| Bluetooth接続先変更（1→2→3のトグル） |
| 1キー| ホイールクリック|
| 2キー| 左クリック |
| 3キー| 右クリック |
